To install dependencies, open a terminal and run :

```
python3 -m pip install -r requirement.txt
```


To start the project :

```
python3 manage.py runserver 8000
```


To make migrations :

```
python3 manage.py makemigrations
```

To apply migrations :

```
python3 manage.py migrate
```
